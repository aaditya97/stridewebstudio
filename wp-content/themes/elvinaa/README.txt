== Description ==
Elvinaa is simple, clean & elegant blog template for bloggers. Elvinaa is crafted with a user friendly approach that has a minimalist look. It’s perfect for Freelancers, Writers, Photographers, Bloggers, Magazine, News Editorial etc. Some of the Premium Features include Featured Slider, Left & Right Sidebars, Social Sharing, Footer Widgets etc.

== Important Note ==
Elvinaa theme supports menu level of depth 3 only.
